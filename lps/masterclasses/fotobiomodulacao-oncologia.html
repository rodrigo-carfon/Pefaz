Para quem é esta masterclass

Conteúdo para toda a equipe oncológica

A fotobiomodulação no contexto oncológico é uma ferramenta multiprofissional. Esta masterclass foi desenvolvida para profissionais de diferentes áreas que atuam ou desejam atuar no suporte ao paciente com câncer.

Odontologia Fisioterapia Enfermagem Medicina Fonoaudiologia Biomedicina Nutrição Farmácia Psicologia

"O cuidado ao paciente oncológico exige integração entre diferentes especialidades. Esta masterclass fornece a linguagem científica comum para que equipes multiprofissionais atuem de forma coordenada e segura."
